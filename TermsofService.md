### **1. Description of Service**

Galactic Campaigns is a utility bot for Discord designed to function as a "Server Architect" for tabletop roleplaying game (TTRPG) communities. Its primary purpose is to automate the administrative setup of new game campaigns by creating channels, roles, and permissions based on administrator-defined templates.

The bot's features include, but are not limited to:
* **Game Template Engine:** A system for server administrators to create, manage, and customize blueprints for new game spaces.
* **Automated Game Creation:** User-initiated creation of Discord channels, categories, and roles based on the server's templates.
* **Permission Management:** Automated configuration of channel and role permissions for Game Masters (GMs), players, and other bots.
* **Moderation Tools:** A configurable approval queue for new game requests to give server staff control over game creation.
* **Premium Features:** Optional, paid features available through a subscription service (e.g., Patreon) that unlock enhanced capabilities like unlimited game creation, advanced management dashboards, and other automation tools.
